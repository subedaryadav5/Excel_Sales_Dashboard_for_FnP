# Excel_Sales_Dashboard_for_FnP
Created a Sales Analysis Dashboard for an online gifting business (FNP) built using Excel. Analyzed 1,000 orders to track revenue trends, customer spending, product and category performance, seasonal demand, delivery efficiency, and regional sales insights to support data-driven decisions.”
ales Analysis Dashboard – FNP (Excel / Power BI)
Project Overview

This project focuses on analyzing sales performance data for an online gifting and floral retail business (FNP). The goal of the project is to convert raw sales data into meaningful insights using an interactive and visually driven dashboard, enabling data-driven decision-making for business stakeholders.

The dashboard provides a holistic view of revenue trends, customer behavior, product performance, and regional demand, helping identify growth opportunities and operational improvements.

 Objectives

Analyze overall sales and revenue performance

Identify high-performing products, categories, and occasions

Understand customer purchasing behavior and spending patterns

Track delivery efficiency and order fulfillment timelines

Discover seasonal and regional sales trends


Tools & Technologies Used

Extract Data - Power Query Editor
Data Cleaning - Power Query Editor
Data modelling - Power Pivot
Data Analysis - Pivot Tables & Measures
Dashbiard - Charts
